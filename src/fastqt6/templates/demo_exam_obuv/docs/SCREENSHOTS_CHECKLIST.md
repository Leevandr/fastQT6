# Скриншоты для DOCX перед сдачей

В ТЗ требуется docx со скриншотами корректной работы системы. Перед сдачей открой Word/LibreOffice и вставь туда эти экраны.

| № | Скриншот | Что должно быть видно |
|---:|---|---|
| 1 | Окно входа | Логин, пароль, кнопка входа, гостевой вход |
| 2 | Неверный вход | Сообщение об ошибке |
| 3 | Гость | Каталог товаров без поиска, фильтра, сортировки и CRUD |
| 4 | Клиент | ФИО клиента, каталог без CRUD |
| 5 | Менеджер | Поиск, фильтр по поставщику, сортировка, вкладка заказов |
| 6 | Администратор | Кнопки добавления, редактирования и удаления товаров |
| 7 | Скидочный товар | Старая цена зачеркнута, новая цена показана |
| 8 | Товар без остатка | Отдельная подсветка строки |
| 9 | Форма товара | Все поля товара, фото, выпадающие списки |
| 10 | Выбор изображения | Фото отображается, приложение не падает |
| 11 | Список заказов | Артикул, статус, пункт выдачи, даты |
| 12 | Форма заказа | Артикул, статус, пункт выдачи, дата заказа, дата выдачи |
| 13 | Запрет удаления товара из заказа | Предупреждение о невозможности удаления |
| 14 | Выход | Возврат на окно входа |

Название файла можно сделать простым: `screenshots.docx`.
